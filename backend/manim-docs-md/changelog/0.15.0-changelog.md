### Breaking changes[¶](#breaking-changes "Link to this heading")

* [#2476](https://github.com/ManimCommunity/manim/pull/2476): Improved structure of the [`mobject`](../reference/manim.mobject.mobject.html#module-manim.mobject.mobject "manim.mobject.mobject") module
  :   Arrow tips now have to be imported from `manim.mobject.geometry.tips` instead of `manim.mobject.geometry`.
* [#2387](https://github.com/ManimCommunity/manim/pull/2387): Refactored [`BarChart`](../reference/manim.mobject.graphing.probability.BarChart.html#manim.mobject.graphing.probability.BarChart "manim.mobject.graphing.probability.BarChart") and made it inherit from [`Axes`](../reference/manim.mobject.graphing.coordinate_systems.Axes.html#manim.mobject.graphing.coordinate_systems.Axes "manim.mobject.graphing.coordinate_systems.Axes")
  :   + [`BarChart`](../reference/manim.mobject.graphing.probability.BarChart.html#manim.mobject.graphing.probability.BarChart "manim.mobject.graphing.probability.BarChart") now inherits from [`Axes`](../reference/manim.mobject.graphing.coordinate_systems.Axes.html#manim.mobject.graphing.coordinate_systems.Axes "manim.mobject.graphing.coordinate_systems.Axes"), allowing it to use [`Axes`](../reference/manim.mobject.graphing.coordinate_systems.Axes.html#manim.mobject.graphing.coordinate_systems.Axes "manim.mobject.graphing.coordinate_systems.Axes")’ methods. Also improves [`BarChart`](../reference/manim.mobject.graphing.probability.BarChart.html#manim.mobject.graphing.probability.BarChart "manim.mobject.graphing.probability.BarChart")’s configuration and ease of use.
      + Added [`get_bar_labels()`](../reference/manim.mobject.graphing.probability.BarChart.html#manim.mobject.graphing.probability.BarChart.get_bar_labels "manim.mobject.graphing.probability.BarChart.get_bar_labels") to annotate the value of each bar of a [`BarChart`](../reference/manim.mobject.graphing.probability.BarChart.html#manim.mobject.graphing.probability.BarChart "manim.mobject.graphing.probability.BarChart").
